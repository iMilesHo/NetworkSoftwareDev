# History

- 1970s: C language, Dennis Ritchie, (Unix operating system)
- 1979: C with Classes, Bjarne Stroustrup
- 1983: name changed to C++
- 1989: first commercial release
- 1998: C++98 standard
- 2003: C++03 standard
- 2011: C++11 standard
- 2014: C++14 standard
- 2017: C++17 standard
