# Initialization in C++

```cpp
int num{}; // zero initialization
int num{100}; // direct initialization
int num = 100; // copy initialization
int num = {100}; // uniform initialization
int num = int{100}; // uniform initialization with type specified
int num(100); // functional form of initialization
int num = int(100); // functional form of initialization
int num = (100); // functional form of initialization
```
